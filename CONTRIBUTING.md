# Contributing to Claude Security Auditor Skill

We welcome contributions to improve the system prompt or add more examples!

## How to Contribute

1.  **Improve the Prompt**: If you find ways to make the JSON report more accurate or the analysis deeper, feel free to suggest changes to `system_prompt.txt`.
2.  **Add Examples**: Add new vulnerable code snippets to the `examples/` directory along with expected JSON reports.
3.  **Refine Documentation**: Help improve the README or other documentation.

## Security Note

This tool is for educational and auditing purposes. Always ensure you have permission to audit any code you provide to Claude.
